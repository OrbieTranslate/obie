ฟีเจอร์ที่ทำได้:

📝 พิมพ์ข้อความ แปล อังกฤษ ⇄ ไทย (สลับทิศทางได้) 🎤 พูดผ่านไมค์ ใช้ Web Speech API ของเบราว์เซอร์แปลงเสียงเป็นข้อความ แล้วแปลให้อัตโนมัติ 🔊 ฟังเสียงคำแปล ใช้ Text-to-Speech อ่านออกเสียงคำแปล (เลือกได้ทั้งเสียงอังกฤษ/ไทย) ใช้ API แปลภาษาฟรีที่ไม่ต้องขอ API key (เช่น MyMemory Translation API)

🔤 พิมพ์ข้อความ แปล อังกฤษ ⇄ ไทย เป็นค่าเริ่มต้น พร้อมสลับภาษาได้ด้วยปุ่มลูกศรตรงกลาง 🌐 เลือกภาษาอื่นเพิ่มได้: จีน เวียดนาม รัสเซีย ญี่ปุ่น เกาหลี ฝรั่งเศส สเปน เยอรมัน 🎤 แท็บสนทนาด้วยเสียง — แตะไมค์ที่ลูกโลกเรืองแสงตรงกลาง (ดีไซน์ตามภาพที่ 1/3) เวลาฟังจะมีวงแหวนเป็นคลื่นออกมา และเปลี่ยนสี/จังหวะตอนกำลังพูดคำแปล พร้อมโชว์บทสนทนาแบบแชทบับเบิลด้านล่าง 🔊 ปุ่มฟังเสียงคำแปล + ปุ่มคัดลอก ในโหมดพิมพ์ข้อความ ใช้ Google Translate endpoint แบบฟรีไม่ต้องมี API key (แม่นยำใกล้เคียง Google Translate จริง) พร้อมระบบสำรอง MyMemory หากเรียกไม่สำเร็จ

หมายเหตุการใช้งาน:

ฟีเจอร์ไมค์ทำงานได้ดีที่สุดบน Chrome (ต้องอนุญาตไมโครโฟน) เปิดไฟล์นี้ในเบราว์เซอร์ได้เลย ไม่ต้องติดตั้งอะไรเพิ่ม
